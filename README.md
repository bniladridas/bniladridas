## 🚀 Quick Start  
Clone & run the project:  
```bash
git clone https://github.com/bniladridas/bniladridas.git  
cd bniladridas  
npm install  
npm run dev  
```  
Then, visit `http://localhost:3000`.  

## 🌍 Deployment  
Deploy easily with **Vercel**:  
```bash
npm install -g vercel  
vercel login  
vercel  
```
